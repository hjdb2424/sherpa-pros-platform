Scope All three flows — Client Find Pros, Pro Find Jobs, Emergency Dispatch
Map Library Google Maps only — remove Mapbox entirely
Layout Hybrid — full-screen + bottom sheet on mobile, split panel on desktop
Markers Progressive Detail — clusters → avatars → mini-cards based on zoom
Emergency UX Rich bottom sheet — ETA, progress bar, pro details, call/message
Theme System preference — respects light/dark mode, no forced override
