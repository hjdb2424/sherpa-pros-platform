Pro Checklist + Materials + Payments — Full System Flow

Review the end-to-end flow. Does this capture what you described?

Complete Flow: Job Accepted → Materials → Work → Payment
1

Checklist Generation (Wiseman)

Scope
Project scope, requirements, code compliance from Code + Architect Wiseman
Process
Step-by-step installation guide, suggested working order, safety notes
Checklist
Phase-by-phase inspection checklist, OSHA compliance, quality gates
Materials List
Auto-generated from Assembly + Pricing Wiseman. Every item, quantity, spec.
2

Pro Reviews + Adjusts Materials

Pro can:
  • Add items (with notes explaining why)
  • Remove items (with notes explaining why)
  • Adjust quantities
  • Swap brands/specs
Wiseman reviews changes:
  • ✅ Approved — change makes sense
  • ⚠️ Warning — recommends alternative
  • ❌ Flagged — safety/code concern
SSP (AI) monitors changes. Flags anything unusual to human SSP for review.
3

Real-Time Pricing (SerpApi → Home Depot)

For each item:
SerpApi queries Home Depot by SKU/name + client zip code → returns real price, stock status, nearest store with availability, product image
Client sees:
Itemized materials list with real prices, total cost, nearest HD store, estimated availability. No surprises.
4

Client Approves + Funds Materials

Option A: Card Hold
Stripe PaymentIntent with capture_method: 'manual'. Funds reserved, not charged yet. 7-day hold.
Option B: Wisetack Financing
Client applies in-app. Real-time approval. $500-$25K. Wisetack pays Sherpa Pros, client pays Wisetack monthly.
Pro Side: Stripe Capital
Pro needs cash flow? Stripe Capital advances funds against future earnings. Pro repays from job payouts.
SCP (AI) guides the client through options. If subscribed to Sherpa Plan, human SCP handles personally.
5

Order Placement + Delivery (Zinc API)

🏪
Pro Pickup
BOPIS order, pro grabs at HD
~30 min
🚛
HD Delivery
Home Depot ships to job site
1-3 days
🚗
Gig Delivery
Uber/DoorDash picks up at HD
1-2 hrs (emergency)
⚖️
Size Check
Wiseman checks weight/size for gig eligibility
<50 lbs → gig OK
6

Work In Progress + Change Orders

Checklist Progress:
Pro checks off tasks as completed. Photos required per phase. SSP monitors progress. Client can view real-time.
Need more materials?
< 10% budget: Auto-approved, client notified
> 10% budget: Client approval required via SCP
Wiseman reviews all additions for code/safety
7

Completion → Charge → Pay Pro

Checklist 100%
All tasks complete, photos verified, client signs off
Card Captured
Stripe captures the held amount. Or Wisetack finalizes the loan.
Pro Paid
Commission deducted, remainder to pro via Stripe Connect. Stripe Capital repaid if used.
