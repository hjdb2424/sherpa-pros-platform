## Three Capture Strategies

### Strategy 1: Emergency First Responder Network (NOW)
- Capture the moment of panic — fastest response wins 80%+ of emergency jobs
- Emergency Ready Pros on retainer ($200-500/mo) guarantee 4hr SLA
- App must already be installed — small jobs are the customer acquisition funnel
- BldSync powers: Dispatch Wiseman (speed), Pricing Wiseman (instant estimates), Checklist Wiseman (IICRC compliance)

### Strategy 2: Insurance Supplement Weapon (Month 3)
- Pricing Wiseman audits Xactimate estimates (15-30% below market)
- Auto-generates data-backed supplements with regional market comparisons
- Contractors get paid what the work actually costs — platform loyalty
- Xactimate .ESC file import → line-item comparison → supplement export

### Strategy 3: Become the MRN (Month 12)
- 200+ IICRC-certified Pros, documented SLA compliance, real-time dispatch
- Pitch regional TPAs: Crawford, Sedgwick, Alacrity, Pilot Catastrophe
- Plug-in network that's faster and more transparent than existing MRNs

## Emergency Dispatch Flow

```
Client taps EMERGENCY
→ Category: Water | Fire | Storm | HVAC | Electrical | Gas | Structural
→ Severity: Critical (immediate danger) | Urgent (damage spreading) | Same-day
→ Auto-detect location OR enter address
→ Photos optional (but encouraged)

Dispatch Wiseman:
→ Filters: Emergency Ready Pros + matching trade + within 60min drive
→ Scores: rating, distance, response history, IICRC certs
→ Critical: top 3 notified simultaneously, first accept wins
→ Urgent: top Pro gets 5min, then cascade
→ Same-day: top 3 notified, 1hr window

Pro accepts:
→ Client sees: real-time ETA, Pro photo, rating, certifications
→ Masked communication channel opens
→ IICRC mitigation checklist auto-generated
→ Emergency pricing displayed (Pricing Wiseman range)

On-site:
→ Pro documents: arrival time, initial photos, moisture readings
→ Mitigation begins (water extraction, board-up, tarping)
→ Drying log tracking (for insurance documentation)
→ Completion photos

Post-mitigation:
→ Client prompted: "File an insurance claim?"
→ YES: Platform generates insurance-ready documentation package
  - Timeline of events
  - Before/during/after photos
  - Moisture readings + drying logs
  - IICRC-compliant mitigation report
  - Pricing Wiseman estimate for full restoration
→ Sherpa Pros manages full restoration as separate job (bid process)
```

## Insurance Documentation Package

When a client files a claim after emergency work, the platform auto-generates:

1. **Incident Timeline** — timestamps from emergency tap to mitigation complete
2. **Photo Documentation** — before/during/after, tagged by room/area
3. **Moisture Mapping** — readings by location over time (for water damage)
4. **Mitigation Report** — IICRC-compliant scope of emergency work
5. **Cost Documentation** — itemized with market rate backing (Pricing Wiseman)
6. **Estimate for Full Restoration** — Pricing Wiseman scope, formatted for adjuster review
7. **Permit Requirements** — Code Wiseman flags for the repair phase

## Xactimate Integration Path

1. Import .ESC files (Xactimate estimate exports)
2. Parse line items → map to Pricing Wiseman categories
3. Compare each line: Xactimate price vs market rate
4. Flag underpaid items with % deviation + regional data
5. Auto-generate supplement with supporting evidence
6. Export in adjuster-friendly format
