# FW Webb Engagement Runbook

## Hard deadline

**LOI SIGNED by Phase 1 Month 4.**

This runbook works backward from that date. Phyrom is the Single Threaded Owner. Every phase has named handoffs, owner, and gating evidence.

## Phase 1 — Prep + warm-intro mapping (Month 0 → Month 1)

### Goals

- FW Webb HQ org chart confirmed
- Warm-intro paths to CEO / COO / VP Partnerships identified
- Intro pitch deck (`02-intro-pitch-deck.md`) finalized + reviewed by 2 trusted advisors
- Tier 1 branch managers identified by name
- LOI template (`03-pilot-loi-template.md`) reviewed by Sherpa Pros counsel

### Tasks

- [ ] Confirm FW Webb CEO / COO / VP Partnerships names via FW Webb website + LinkedIn (Phyrom)
- [ ] Map warm-intro paths: NE Pro Council, mutual-customer GCs, Sherpa Pros advisor network, FW Webb supplier reps (Phyrom)
- [ ] Draft 3 intro outreach messages: (a) cold-but-warm direct, (b) via warm-intro, (c) via FW Webb supplier rep referral (Phyrom + advisor)
- [ ] Lock intro pitch deck — review with Sherpa advisors + 1 contractor friend who buys at FW Webb (Phyrom)
- [ ] Counsel review of LOI template — flag binding-vs-non-binding sections (Phyrom + counsel)
- [ ] Identify Tier 1 branch managers by name via LinkedIn + FW Webb branch directory (Phyrom)
- [ ] Build 1-pager economic-model summary for branch-manager-level conversations (Phyrom)

### Gating evidence

- FW Webb HQ org chart with names + contact info
- 3 confirmed warm-intro paths
- Pitch deck reviewed and approved
- LOI template counsel-reviewed
- Tier 1 branch managers named

### Exit criteria

Phyrom can credibly request a 30-min meeting with at least one of FW Webb CEO / COO / VP Partnerships.

## Phase 2 — Initial HQ meetings (Month 1 → Month 2)

### Goals

- First meeting with at least one of FW Webb CEO / COO / VP Partnerships
- Buy-in on the strategic logic ("yes, the partnership shape makes sense")
- Regional VP introductions secured
- 5 Tier 1 branch managers introduced

### Tasks

- [ ] Send warm-intro request to first FW Webb HQ exec (Phyrom)
- [ ] Hold first 30-min HQ meeting; share intro deck slides 1-7 only (Phyrom)
- [ ] Follow up within 48 hrs with full deck + economic model + LOI template summary (Phyrom)
- [ ] Schedule 60-min follow-up with VP Partnerships + Regional VP North + Regional VP South (Phyrom)
- [ ] Coordinate Regional VP intros to 5 Tier 1 branch managers (Phyrom + Reg VPs)
- [ ] Send branch-manager intro emails (Phyrom — personalized per branch)

### Gating evidence

- HQ meeting completed; meeting notes circulated
- VP Partnerships engaged as primary day-to-day counterpart
- Regional VP introductions made
- 5 Tier 1 branch manager calls scheduled

### Exit criteria

FW Webb has assigned a named primary counterpart (VP Partnerships) who is actively responding to email within 48 hrs.

### Risks to monitor

- FW Webb leadership references "we want to think about it" → escalate to founder-to-founder ask
- Regional VP says branch managers are too busy → propose virtual intros first, on-site visits second
- HQ asks for exclusivity outside what LOI offers → defer to legal scope; do not concede

## Phase 3 — On-site branch visits (Month 2 → Month 3)

### Goals

- Phyrom physically visits each of 5 Tier 1 candidate branches with Regional VP
- Branch managers see and feel the Sherpa Hub vision
- Branch-level data collected (sq-ft availability, parking, loading-dock fit, current pro counter throughput)
- 5 branch managers verbally support hosting a pilot

### Tasks

- [ ] Schedule 5 on-site visits (Phyrom + Regional VP + Branch Mgr) — half-day each (Phyrom)
- [ ] Bring physical mockup / FFE plan for the 800 sq-ft Sherpa Hub footprint (Phyrom)
- [ ] Walk through the branch with the Branch Mgr; identify the proposed Hub footprint location (Phyrom + Branch Mgr)
- [ ] Document Branch Site Schedule data (sq ft available, configuration, doorways, utilities) per branch (Phyrom + FW Webb facilities)
- [ ] Interview Branch Mgr on: top 50 pro accounts, pro-account churn signals, top SKU velocity, counter-pain-point list (Phyrom)
- [ ] Take photos + measurements; build per-branch buildout estimate refinement (Phyrom)

### Gating evidence

- 5 completed on-site visits with notes
- 5 Branch Site Schedules drafted
- 5 Branch Mgrs verbally supportive
- Per-branch buildout cost refined to within ±10% of $80K planning number

### Exit criteria

5 Tier 1 branches have viable Hub footprints + supportive Branch Mgrs.

If <5 viable, substitute with Tier 2 candidates from `01-target-contact-list.md`.

## Phase 4 — LOI negotiation + legal review (Month 3 → Month 4)

### Goals

- LOI negotiated to executable form
- Both Parties' counsel signed off
- Sub-lease terms finalized per branch
- Definitive Master Services Agreement structure agreed

### Tasks

- [ ] Send LOI template (`03-pilot-loi-template.md`) to FW Webb VP Partnerships + GC (Phyrom)
- [ ] Schedule LOI negotiation working session (Phyrom + Sherpa counsel + FW Webb VP Partnerships + FW Webb GC)
- [ ] Iterate on flagged [LEGAL] sections: §2.1 footprint classification, §3 economics, §6 data sharing, §7 exclusivity, §10 termination, §11 IP, §13 binding/non-binding, §14 governing law (joint counsel)
- [ ] Finalize per-branch Branch Site Schedules (FW Webb facilities + Phyrom)
- [ ] Negotiate any commercial term adjustments (e.g. sub-lease fee, exclusivity scope) with explicit P/L impact analysis from `04-fw-webb-economic-model.md` (Phyrom + FW Webb VP Partnerships)
- [ ] Final review by both CEOs (Phyrom + FW Webb CEO)

### Gating evidence

- LOI text finalized
- Both counsels signed off
- 5 Branch Site Schedules attached as Exhibit B
- CEO sign-off

### Exit criteria

LOI is execution-ready by **Phase 1 Month 4 Week 1** to allow signature in Week 2-3.

- [ ] Joint press release approved + scheduled (Sherpa marketing + FW Webb marketing)
- [ ] Internal kickoff meeting both teams (Phyrom + FW Webb VP Partnerships + ops counterparts)
- [ ] Set up joint Slack channel / shared workspace
- [ ] Schedule first QBR for Pilot M3

- Joint press release published

## Phase 6 — Pilot Hub buildouts begin (Month 4 → Month 6)

### Goals

- 5 pilot Hubs in active buildout
- First Hub opens (Plaistow NH or Manchester NH) by Pilot Month 2 from LOI
- All 5 Hubs operational by Pilot Month 6 from LOI

### Tasks

- [ ] Mirror Wave A Hub #1 buildout playbook adapted for co-located format (Sherpa Network Ops)
- [ ] Stagger the 5 buildouts: Hub 1 (Plaistow) week 1, Hub 2 (Haverhill) week 3, Hub 3 (Manchester) week 5, Hub 4 (Woburn) week 7, Hub 5 (Warwick) week 9 (Sherpa Network Ops)
- [ ] Hire Hub Manager for each Hub on a -6-week lead from open date (Sherpa HR)
- [ ] FFE delivery + installation per Hub buildout playbook
- [ ] Pre-open member acquisition campaign per Hub (FW Webb in-branch signage + counter handouts + Sherpa Marketplace push)
- [ ] Sherpa Materials engine integration per Hub (per spec `05-sherpa-materials-hub-integration-spec.md`)
- [ ] Per-Hub dashboard + Network Ops dashboard wired up (per specs `06` + `07`)
- [ ] Soft-open invite-only beta per Hub for first 2 weeks
- [ ] Public open per Hub
- [ ] Pilot M0 starts on Hub #1 soft-open date

### Gating evidence

- All 5 Hubs operational by Pilot Month 6
- All Hub Managers hired + trained
- Sherpa Materials engine fulfilling orders at all 5 Hubs
- Per-Hub + Network Ops dashboards live
- Pilot M0 confirmed

### Exit criteria

5 Hubs operational; Pilot Month 6 review (per `09-pilot-success-metrics.md` Layer 2 gates) imminent.
