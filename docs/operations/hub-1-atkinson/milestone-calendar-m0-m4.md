## Week-by-week milestones

### M0 — Foundation phase (Weeks 1–4)

**Week 1 — Architect RFP + Hub Manager job posting live**
- *Owner: Phyrom*
- Send RFP to 3 architect firms (Round 1: Warrenstreet, Tennant/Wallace, Winter Holben).
- Post Hub Manager job to LinkedIn + Indeed + 1 NH/MA trade-school placement office.
- Confirm zero-cost colocation terms documented with HJD.

**Week 2 — Architect proposals received + Hub Mgr first interviews**
- *Owner: Phyrom*
- Review 3 proposals; if <2 viable, send Round 2 RFPs to backup firms.
- Phone-screen first 5 Hub Mgr applicants.

**Week 3 — Architect selected + contracted**
- *Owner: Phyrom + selected architect*
- Sign architect AIA B105 (or equivalent fixed-fee small-project agreement).
- Architect kickoff site visit at Atkinson location.
- Hub Mgr second-round interviews (in-person, Atkinson coffee meeting per posting).

**Week 4 — Schematic design complete**
- *Owner: Architect*
- Architect delivers schematic design package (floor plan, MEP narrative, code summary).
- Phyrom reviews + approves.
- Hub Mgr finalist(s) identified; offer prep starts.

### M1 — Permit + GC selection phase (Weeks 5–7)

**Week 5 — Permit drawings stamped + submitted**
- *Owner: Architect → Atkinson AHJ*
- Architect submits stamped permit set to Atkinson Building Department.
- Begin GC RFP process per existing `03-contractor-rfp-and-criteria.md`.
- *CRITICAL CALL-OUT: NH permit review cycles run 10–20 business days; submit early in week to maximize buffer.*

**Week 6 — GC RFP responses due + vendor account opens begin**
- *Owner: Phyrom*
- 3 GC bids received; review + reference-check.
- Open accounts: F.W. Webb, Granite City Electric, Home Depot Pro, regional supply houses.
- Hub Mgr offer extended to top finalist.

**Week 7 — Permit pulled + GC selected**
- *Owner: Atkinson AHJ → Phyrom*
- Building permit issued. (If delayed past W7, escalate to Atkinson Building Inspector directly.)
- Sign GC contract (lump-sum, AIA A105).
- Hub Mgr accepts offer; start date set for W11 (allows 2-week notice + 1-week ramp).
- *CRITICAL CALL-OUT: Permit slip = grand-open slip. If not pulled by end of W7, trigger contingency plan — soft-open moves from W14 to W15.*

### M2 — Construction phase (Weeks 8–11)

**Week 8 — GC mobilization + demo**
- *Owner: GC*
- Site protection, demo per drawings.
- Phyrom on-site Mondays for weekly OAC (Owner-Architect-Contractor) meeting.

**Week 9 — Rough framing + MEP rough-in begins**
- *Owner: GC + MEP subs*
- Framing, electrical rough, plumbing rough, HVAC rough.
- Vendor accounts confirmed; opening inventory POs cut per `06-opening-inventory-list.md`.

**Week 10 — MEP rough complete + inspections + insulation**
- *Owner: GC*
- Rough inspections passed (electrical, plumbing, framing, mechanical).
- Insulation install.
- Marketing: soft-open invite list drafted (target 30–50 names).

**Week 11 — Hub Manager starts + drywall + paint**
- *Owner: Hub Mgr (Day 1) + GC*
- **Hub Mgr Day 1:** Phyrom hands over GC relationship, vendor account list, opening inventory POs.
- Drywall hung + finished; primer coat.
- Hub Mgr begins local pro outreach (target 50+ pros signed up by W16).

### M3 — Finish + equipment phase (Weeks 12–14)

**Week 12 — Finish paint, flooring, casework, signage order**
- *Owner: GC + Hub Mgr*
- Final paint, flooring install, counter casework, lounge millwork.
- Signage ordered (exterior monument, interior wayfinding).
- Press kit drafted (Hub Mgr + Phyrom).

**Week 13 — Equipment install begins + POS setup**
- *Owner: Hub Mgr + vendor techs*
- Counter POS, dispatch workstations, network/Wi-Fi, security cameras.
- Pro lounge furniture delivered + installed.
- Final inspections scheduled.
- *CRITICAL CALL-OUT: POS + dispatch software must be live + tested by end of W13. Any IT slip cascades into soft-open dry runs.*

**Week 14 — CO + soft-open (invite-only)**
- *Owner: Atkinson AHJ → Hub Mgr → Phyrom*
- Final inspections passed; Certificate of Occupancy issued.
- **Soft-open Thursday W14 evening:** 30–50 invited pros, family, friendlies. Free food, full demo, feedback collection.
- Punch-list captured for W15.

### M4 — Open phase (Weeks 15–16)

**Week 15 — Punch list + dry runs + press push begins**
- *Owner: Hub Mgr + GC*
- All punch items resolved.
- Daily dry runs of opening procedures (counter, dispatch, lounge).
- Press releases to NH Business Review, Union Leader, Atkinson local press.
- Final inventory top-up.

**Week 16 — Grand-open `[GRAND-OPEN-DATE-TBD]`**
- *Owner: Phyrom + Hub Mgr*
- Public grand-open day.
- Phyrom on-site full day; Hub Mgr running floor.
- See `pre-open-checklist.md` for week-of execution detail.

---

## Risk reserve

Schedule contains **0 weeks of float** on the critical path. If Week 7 permit pulls slip into Week 8, every downstream milestone slips by the same delta. Recommend Phyrom budget **2 weeks of grand-open slip tolerance** in external comms — never publish a hard grand-open date until W14 CO issuance.
