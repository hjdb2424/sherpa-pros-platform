# Hub #1 Atkinson NH - Architect Engagement Request for Proposal (RFP)

## Purpose

Engage a New Hampshire-licensed commercial architect to produce permit-ready drawings, code-compliance review, fixtures-furnishings-equipment (FF&E) layout, and signage design for the Hub #1 fit-out at HJD Builders LLC HQ in Atkinson NH.

**Engage by:** Phase 1 Month 1 (target signed engagement letter within 30 days of this RFP).

**Budget cap:** $12,000 - $18,000 fixed-fee engagement (all phases below included).

---

## 1. Project Scope

### 1.1 As-built drawings
- [ ] Field-measure existing HJD HQ bay
- [ ] Produce as-built floor plan, reflected ceiling plan, electrical one-line, mechanical schematic
- [ ] Document existing conditions (structural, MEP, finishes)

### 1.2 Code-compliance review
- [ ] Review against IBC (International Building Code) as adopted by NH
- [ ] Review against NH State Building Code amendments
- [ ] Review against Atkinson Town building ordinance + zoning ordinance
- [ ] Review accessibility (ADA + NH state accessibility code)
- [ ] Identify required upgrades (egress, fire separation, accessibility, occupancy load) and cost-impact each

### 1.3 Permit drawings
- [ ] Construction-ready architectural plans
- [ ] Stamped + signed plans for Atkinson Building Department
- [ ] Plans include: floor plan, reflected ceiling plan, demo plan, finish schedule, door/hardware schedule, electrical plan (coordinated with electrical sub), mechanical plan (coordinated with HVAC sub), plumbing plan, life-safety plan
- [ ] Coordinate with engineering subs (structural / MEP) for stamped sub-discipline drawings as required by code

### 1.4 FF&E layout
- [ ] Design furniture, fixtures, equipment layout coordinated with WS1.3 equipment manifest
- [ ] Specify finish schedules (flooring, paint, ceiling, lighting)
- [ ] Specify millwork (POS counter, member-lounge work table, kit-assembly bench)
- [ ] Specify storefront / showroom layout for brand consistency with Sherpa Pros visual identity

### 1.5 Signage design
- [ ] Design exterior primary sign + secondary signage per Atkinson sign ordinance
- [ ] Design interior wayfinding system
- [ ] Coordinate signage drawings for permit submission
- [ ] Source signage fabricator + installation oversight

## 3. Architect Selection - Target 3 Candidates

Identify and shortlist 3 NH-licensed commercial architects with strong contractor / supply-house / retail-light-industrial experience. Recommended search radius: Manchester / Concord / Portsmouth / Salem NH and northern MA border (Andover / Methuen).

### Sourcing criteria
- [ ] NH-licensed (active, in good standing with NH Joint Board of Licensure)
- [ ] 5+ years commercial fit-out experience
- [ ] Demonstrated portfolio: contractor supply, retail, light industrial, or mixed-use commercial
- [ ] Experience with Town of Atkinson or peer NH towns (Plaistow, Hampstead, Salem) - knows local Building Department personalities + permit timing
- [ ] Comfortable with $12-18K fixed-fee engagement at this scope
- [ ] Available to start within 30 days

### Recommended sourcing channels
- [ ] AIA NH Chapter (American Institute of Architects, NH chapter directory)
- [ ] NHHBA member directory (NH Home Builders Association - many member firms have architects on retainer or in-network)
- [ ] LinkedIn search: "commercial architect" + "New Hampshire"
- [ ] Referrals from HJD Builders sub network (electricians, GCs, supply reps often know who they prefer working with)
- [ ] Atkinson Town Building Department - ask Building Inspector who has filed cleanly recently
- [ ] Phyrom's existing professional network from HJD project history

### Shortlist evaluation
- [ ] Initial 30-min phone screen with 5+ candidates
- [ ] Site visit + RFP walkthrough with top 3
- [ ] Reference check: last 2 commercial fit-out clients (timeline adherence, budget adherence, change-order discipline, ease of working with)
- [ ] Insurance + license verification
