# Hub #1 Atkinson NH - Commercial Fit-Out Permit Checklist

## Purpose

Document every permit, inspection, and approval required to legally complete the Hub #1 fit-out at HJD Builders LLC HQ in Atkinson NH and obtain a Certificate of Occupancy (CO) for member access.

**Permit responsibility:** Most permits pulled by GC (per `03-contractor-rfp-and-criteria.md` contract terms). Owner (Phyrom / HJD Builders LLC) responsible for signage permit + zoning sign-off + CO application. Architect (per `02-architect-engagement-rfp.md`) provides stamped drawings for all permit packages.

**Atkinson Town Building Department contact path:** Town Hall, Atkinson NH (Phyrom to confirm exact contact, hours, fee schedule on first call - small NH town, often part-time staff with limited windows).

---

## 1. Building Permit (Primary)

**Pulled by:** GC (with owner co-signature)
**Authority:** Town of Atkinson Building Department
**Purpose:** Authorize structural / non-structural alterations to existing commercial space.

### Required submission
- [ ] Building permit application (Atkinson Town form)
- [ ] Architect-stamped construction drawings (full set per `02-architect-engagement-rfp.md`)
- [ ] Code-compliance memo from architect
- [ ] Site plan (existing + proposed conditions)
- [ ] Owner authorization letter (HJD Builders LLC authorizing GC to pull permit on owner's behalf)
- [ ] GC license + insurance
- [ ] Fee payment

### Estimated fee
- [ ] $1,500 - $4,500 (typical NH commercial permit fee, scaled to construction value 1-3% of build cost)

### Lead time
- [ ] 3-4 weeks from submission to issuance (typical Atkinson commercial review timeline)
- [ ] Add 2-week buffer for first-comment cycle if Building Inspector flags items needing revision

### Inspections triggered
- [ ] Pre-construction / footing inspection (if structural)
- [ ] Framing inspection
- [ ] Insulation inspection
- [ ] Drywall / pre-finish inspection
- [ ] Final building inspection (precondition for CO)

## 2. Electrical Permit

**Pulled by:** Licensed electrician (sub to GC)
**Authority:** Town of Atkinson (or NH State Electrical Inspector if Atkinson defers to state)
**Purpose:** Authorize all electrical work including panel upgrade, new circuits, lighting, low-voltage.

### Required submission
- [ ] Electrical permit application
- [ ] Electrical plan from architect (or sub-engineer if separately stamped)
- [ ] Master electrician license + insurance
- [ ] Fee payment

### Estimated fee
- [ ] $500 - $1,800 (scaled to circuit count + panel work)

### Lead time
- [ ] 1-2 weeks from submission

### Inspections triggered
- [ ] Rough electrical (pre-drywall)
- [ ] Service / panel inspection (if upgrade)
- [ ] Final electrical (precondition for CO)

---

## 5. Signage Permit

**Pulled by:** Owner (Phyrom / HJD Builders LLC) or signage fabricator
**Authority:** Town of Atkinson - Planning + Building combined (sign ordinance enforcement)
**Purpose:** Authorize exterior primary sign + secondary signs per Atkinson sign ordinance.

### Required submission
- [ ] Signage permit application
- [ ] Sign drawings (size, materials, mounting, illumination, setback)
- [ ] Site plan showing sign location relative to property line + setback compliance
- [ ] Compliance statement against Atkinson sign ordinance (size limits, illumination limits, number-of-signs limits)
- [ ] Fee payment

### Estimated fee
- [ ] $200 - $600

### Lead time
- [ ] 2-3 weeks (Planning Board review may be required for illuminated or larger signs)

### Inspections triggered
- [ ] Sign installation inspection (electrical hookup verification if illuminated)

---

## 6. Fire Department Approval

**Pulled by:** GC (typically routed through Building Permit submission)
**Authority:** Atkinson Fire Department
**Purpose:** Verify life-safety compliance - egress, fire separation, fire extinguishers, exit signage, alarm panel (if required), sprinkler (if required).

### Required submission
- [ ] Life-safety plan from architect
- [ ] Occupancy load calculation
- [ ] Fire-extinguisher locations
- [ ] Exit-signage plan
- [ ] If sprinkler system existing: confirm coverage adequate; if new system required: separate sprinkler permit + fire protection engineer involvement (additional $5-12K cost if sprinkler retrofit needed)

### Estimated fee
- [ ] $200 - $800 (review fee bundled with building permit in many NH towns)

### Lead time
- [ ] Concurrent with building permit (3-4 weeks)

### Inspections triggered
- [ ] Pre-CO fire inspection (precondition for CO)

---

## 7. Health Department Approval (Conditional)

**Pulled by:** Owner if applicable
**Authority:** Atkinson Health Officer / Rockingham County Health
**Purpose:** Required only if Hub serves food beyond pre-packaged / member-lounge coffee.

### Trigger
- [ ] Member-lounge coffee + pre-packaged snacks: typically NO permit required
- [ ] If serving prepared food at training events: food-service permit required ($150-$400, 2-week review)

### Recommendation
- [ ] Phase 1: NO food prep on site. Coffee + bottled drinks + sealed snacks only. Avoid Health Department permit path entirely.
- [ ] Training-event catering: cater from licensed local restaurants (cost passed through to event sponsors)

---

## 8. Sprinkler / Fire Suppression Permit (Conditional)

**Pulled by:** Fire protection sub
**Authority:** Atkinson Fire Department + State Fire Marshal
**Purpose:** Required only if existing sprinkler system requires modification OR if occupancy classification triggers new sprinkler requirement.

### Trigger
- [ ] If HJD HQ already sprinklered + Hub footprint coverage adequate: minor permit ($300-$800)
- [ ] If Hub footprint requires sprinkler retrofit: $8-$25K additional cost + 6-week design + permit + install timeline

### Recommendation
- [ ] Architect to confirm during code-compliance review (per `02-architect-engagement-rfp.md` deliverable 1.2)

---

## 9. Certificate of Occupancy (CO)

**Pulled by:** Owner (Phyrom)
**Authority:** Town of Atkinson Building Department
**Purpose:** Final authorization to occupy + operate the Hub.

### Preconditions (all required before CO issuance)
- [ ] Final building inspection passed
- [ ] Final electrical inspection passed
- [ ] Final plumbing inspection passed
- [ ] Final mechanical inspection passed
- [ ] Fire inspection passed
- [ ] Signage inspection passed (if applicable to CO timing)
- [ ] All permits closed out (no open inspections)
- [ ] Architect's affidavit of completion (in some NH jurisdictions)

### Required submission
- [ ] CO application
- [ ] Confirmation all final inspections passed
- [ ] Fee payment

### Estimated fee
- [ ] $100 - $400

### Lead time
- [ ] 1-2 weeks after all final inspections passed

## 12. Critical-Path Risks + Mitigations

| Risk | Mitigation |
|---|---|
| Atkinson Building Inspector unavailable for inspection within 48 hours of GC request | Build 5-day inspection buffer into GC schedule; offer Friday inspection slots |
| Sprinkler retrofit triggered late in design (after permit submission) | Architect flag in code-compliance memo (week 3 of architect engagement) before permit submission |
| Signage Planning Board review required (illuminated sign) | Submit signage permit in parallel with building permit (do not wait until end) |
| Zoning question late-discovered (e.g., parking count below ordinance) | Confirm zoning compliance in `01-real-estate-confirmation.md` before architect engagement |
| Final inspection failure on minor item delays CO | Architect or GC walk-the-job 1 week before final inspection request to pre-flight |
