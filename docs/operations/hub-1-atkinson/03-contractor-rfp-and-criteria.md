# Hub #1 Atkinson NH - General Contractor RFP + Selection Criteria

## Purpose

Engage an arms-length General Contractor (GC) to execute the Hub #1 tenant-improvement build-out. Per the Franchise Model spec's franchise-readiness requirement, Phyrom / HJD Builders LLC must NOT serve as the GC for this project - the build must be at arms-length so the per-Hub buildout cost basis is defensible to prospective franchisees in FDD Item 7.

**Bid out:** Phase 1 Months 1-2
**Select:** Phase 1 Month 2
**Build:** Phase 1 Months 2-3
**Soft-open:** Phase 1 Month 3 (target)

**Budget range:** $85,000 - $185,000 all-in build-out (excluding equipment per `05-equipment-manifest.md` and inventory per `06-opening-inventory-list.md`).

---

## 1. Why Arms-Length GC (Not HJD Builders)

Per the parallel Franchise Model spec, the Hub #1 cost basis must be documentable + defensible in the FDD. If HJD Builders is the GC, the FDD disclosure becomes "founder's own construction company built it at internal pricing" - that's neither a credible market price for prospective franchisees nor a defensible documented capex line item.

Engaging an arms-length GC produces a third-party invoice trail that:
- Establishes a real-market cost basis for FDD Item 7 (initial investment range)
- Validates the spec section 9 capex model with actual third-party numbers
- Removes founder-conflict appearance for franchisee due diligence
- Provides a benchmark against which Hub #2 build-out (FW Webb co-location at lower scope) can be compared

## 2. Scope of Work

### 2.1 Site preparation + demolition
- [ ] Demo existing partition walls (per architect plan)
- [ ] Demo existing flooring (if not retained)
- [ ] Remove existing fixtures + equipment not retained for Hub use
- [ ] Cap or relocate existing MEP services as required

### 2.2 Tenant-improvement build-out
- [ ] Frame new partition walls per architect plan (warehouse / showroom / lounge / training / restroom separations)
- [ ] Flooring: epoxy or polished concrete in warehouse zones, LVT or polished concrete in showroom + lounge, carpet tile in training room
- [ ] Paint per finish schedule
- [ ] Drop ceiling or exposed ceiling per architect plan + lighting design
- [ ] Doors + hardware (member entry keypad/badge ready, dock door, restroom, internal partitions)
- [ ] Storefront window (if retrofit to existing facade required for member-entry visibility)

### 2.3 Electrical
- [ ] Panel upgrade (if 200A->400A required per real-estate memo section 3)
- [ ] 6 dedicated 20A circuits per real-estate memo
- [ ] LED lighting throughout: warehouse (200-400 lux), showroom + lounge (300-500 lux), training (500 lux), tool cage (400 lux), exterior security
- [ ] Emergency / egress lighting per code
- [ ] Electrical drops at POS counter, kiosks, training A/V, kit-assembly bench, security NVR location, member-lounge appliances

### 2.4 HVAC modifications
- [ ] Add zone control if existing HVAC is single-zone (per real-estate memo section 3)
- [ ] Mini-split add to training room or member lounge if existing HVAC capacity insufficient
- [ ] Confirm warehouse zone holds 55F minimum in NH winter (inventory freeze protection)

### 2.5 Plumbing
- [ ] Sink in receiving / kit-assembly area (handwash + parts cleanup)
- [ ] Restroom ADA renovation if existing restroom non-compliant
- [ ] Hot water service to sink + restroom

### 2.6 Security wiring
- [ ] Pre-wire 8 IP camera locations per equipment manifest
- [ ] Pre-wire NVR location with conditioned power + cooling
- [ ] Pre-wire keypad + badge reader at member entry
- [ ] Pre-wire alarm panel + door/window contacts
- [ ] Run Cat6A to all camera + access-control + POS + kiosk locations

### 2.7 Network / IT pre-wire
- [ ] Cat6A backbone to POS counter, 2 customer-facing kiosks, training A/V, manager workstation, 4 staff workstations
- [ ] Wi-Fi access point pre-wire locations (3 APs minimum: warehouse, showroom + lounge, training)
- [ ] Conduit + low-voltage backboxes per architect plan

### 2.8 Signage installation
- [ ] Exterior primary sign install (mounting structure + electrical for illumination)
- [ ] Building-mounted secondary signs
- [ ] Interior wayfinding install

### 2.9 Final
- [ ] Punch-list resolution
- [ ] Final cleaning
- [ ] Final inspections + Certificate of Occupancy hand-off
- [ ] One-year workmanship warranty on all installed work

## 3. Selection Criteria + Scoring Matrix

| Criterion | Weight | Notes |
|---|---|---|
| NH GC license + insurance + bonding capacity | Pass / fail | Mandatory |
| Relevant tenant-improvement portfolio (commercial fit-out, 1,500-5,000 sq ft) | 20% | Show last 5 similar jobs |
| Subcontractor bench depth (in-house electrician, plumber, HVAC sub relationships) | 15% | Reduces coordination risk |
| Schedule discipline (on-time completion last 5 jobs) | 15% | Reference-check verifiable |
| Budget discipline (change-order rate <5% on last 5 jobs) | 15% | Reference-check verifiable |
| Bid competitiveness | 15% | Within $85-185K range |
| Town of Atkinson / Rockingham County permit experience | 10% | Knows local inspectors |
| Cultural fit (founder-friendly, transparent, communicative) | 10% | Phyrom impression after on-site walk |

## 6. Contract Structure

- [ ] AIA A101 stipulated-sum contract
- [ ] AIA A201 general conditions
- [ ] Liquidated damages: $500/day after substantial completion target date (negotiable)
- [ ] Retainage: 10% held to final completion + punch-list closeout
- [ ] Change-order procedure: written CO required, 7-day GC review, owner approval before work commences (no verbal COs)
- [ ] Insurance: GC names HJD Builders LLC + Sherpa Pros LLC as additional insured on general liability + umbrella
- [ ] Lien waivers: GC + all subs provide partial lien waivers with each draw, final lien waivers at retainage release
- [ ] Warranty: 1-year workmanship, manufacturer warranties pass-through

## 8. Budget Range Justification

Per integration spec section 9, Hub #1 capex floor is ~$60K because of HJD HQ zero-cost colocation. The $85-185K range here covers:

| Build-out scope | Low | High |
|---|---|---|
| Demo + framing + finishes (1,800-2,400 sq ft) | $35K | $75K |
| Electrical (panel upgrade if needed + circuits + lighting) | $20K | $45K |
| HVAC modifications | $5K | $20K |
| Plumbing | $5K | $15K |
| Security + low-voltage pre-wire | $8K | $15K |
| Signage installation (fabrication separate per architect) | $5K | $12K |
| Permits + inspections + bonds | $3K | $8K |
| GC overhead + profit (15-20%) | $4K | $-included |
| **Total** | **$85K** | **$185K** |

Selected GC's bid will land within this range. If lowest bid is below $85K, scrutinize for missing scope / underbidding risk. If highest bid is above $185K, scrutinize for over-spec or non-competitive pricing.
