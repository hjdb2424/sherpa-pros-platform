Pillar 1 — The Problem

Lead-gen platforms waste contractors' money and fail homeowners on licensed work. Angi shows 41% service-quality inconsistency and is BBB un-rated. Thumbtack has reported deposit-theft losses of $300–$1,600 per incident. TaskRabbit charges 35% in fees and doesn't cover licensed trades. Handy has no code intelligence at all. (per spec §2.2)

The pressure is intensifying: Massachusetts has the largest construction labor shortage in the nation — 2.5% construction unemployment (lowest in 17+ years), 30% of MA residential construction workers are 55+, and project completion times have stretched from 7 to 11 months. (per spec §2.1) The licensed trades the lead-gen platforms can't filter for are exactly the ones in shortest supply.

The "licensed + code-aware + true marketplace" quadrant is empty. We plant the flag there. (per spec §2.2)

Pillar 2 — The Solution and the Moat

Sherpa Pros: jobs, not leads. Licensed only. Code-aware. Permit-aware. Rebate-aware. Web at www.thesherpapros.com, iOS via TestFlight, PWA on Android. Built on Next.js 16, Stripe Connect, Neon Postgres + PostGIS, Clerk. (per CLAUDE.md)

  • Take rate, not lead-share — 5% during beta, 10% at scale, roughly half a contractor's all-in Angi cost. (per spec §5.2)
  • License + insurance verified at onboarding, re-checked on renewal.
  • Code-aware quote validation runs every quote against NEC, IRC, MA Electrical, NH state code, and surfaces missing line items, wrong materials, and code violations before the homeowner sees the quote. (per spec §1)
  • Permit-aware and rebate-aware — knows which jobs need a Boston ISD permit, which qualify for Mass Save / National Grid / EV-charger rebates, and walks both sides through it. (per spec §2.3)
  • Stacked revenue lines: marketplace take rate (5–12% labor commission floor), + Sherpa Materials coordination fee (8–12%, stacks on top of labor), $49/month Pro subscription (recurring), Property Manager tier at $4 → $1.50 per unit/month (enterprise upside), and Success Manager retainer (Sherpa Manager tier) → blended 18–25% per job all-in. (per spec §3.2 + §5.2 + materials thesis footnote)
  • Six platform pillars shipped this quarter: Sherpa Threads (in-app chat + text-message bridge) · Sherpa Smart Scan (optical character recognition; auto-tags Schedule C + Capital Expenditure / Operating Expense) · Sherpa Mobile (iOS in TestFlight, Android via Expo) · Sherpa Guard (Role-Based Access Control + audit logs — SOC 2-readiness signal) · Sherpa Dispatch (multi-trade timeline orchestration across 6+ trades) · Sherpa Materials (internal materials engine + Zinc API ordering + Uber Direct same-day delivery). Plus the rest of the surface — Sherpa Marketplace · Hub · Home · Manager · Score · Rewards · Flex — and a public /flex landing page + 12-capability splash now live.

The moat is the code-intelligence layer. It is not licensable, not easily cloneable, and required a working GC plus a multi-year codebase to build. National lead-gen platforms cannot copy it without a structural rebuild.
