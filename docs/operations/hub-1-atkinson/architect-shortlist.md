# Architect Shortlist — Hub #1 Atkinson, NH Fit-Out

**Project profile:** ~3,000–4,000 sq ft interior fit-out of existing structure on HJD HQ campus, Atkinson NH. Mixed-use: dispatch desk + counter retail + pro lounge + small back-of-house. Light-commercial scope, NH state building code + Atkinson local AHJ. Target: drawings issued for permit by Phase 1 Week 4–5, CO before Phase 1 Week 14.

**Selection criteria:** NH-licensed (or NCARB reciprocal to NH), prior light-commercial fit-out work in the 2K–8K sq ft range, comfortable with fast-track schedules, available to engage in M0.

2. Confirm firm currently operating + principal contact via firm website + LinkedIn.
3. Confirm 1+ comparable project (light-commercial fit-out, 2K–8K sq ft) completed in last 36 months — use firm's project portfolio page.
4. Get principal contact email (avoid info@ catchall) — LinkedIn often works for this.
